# Go Project - Rest API

This is a Go-based REST API project. Follow the instructions below to set up and run the application.

## Clone the Repository

```sh
git clone https://github.com/ESWARVETLA-19/Rest_API_GO.git
cd Events_Handler
```

## Build the Project

```sh
go build -o app
```

## Run the Project

```sh
go run main.go
```
